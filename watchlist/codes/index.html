Codes

This page contains all the codes used to categorize movies, dramas, anime, web series, and other content across the site.

ⓘ Note:

These codes are used to organize content based on the original country and language of production — not the language in which the content was watched.

  • CD = Chinese Donghua
  • KHA = Korean Hanguk Aeni
  • AA = American Animation
